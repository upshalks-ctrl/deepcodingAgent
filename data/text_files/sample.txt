# 示例文本文档

这是一个用于测试文档处理器的示例文件。

## 第一部分：概述

本文档展示了文本处理器的功能，包括：
- 自动编码检测
- 段落分块
- 元数据提取

文本处理器能够自动识别文件编码格式，包括UTF-8、GBK、GB2312等。

## 第二部分：技术特性

### 编码支持
文本处理器支持多种编码格式：
1. UTF-8（默认）
2. GBK
3. GB2312
4. Latin-1

### 分块策略
根据文档类型，文本处理器采用不同的分块策略：
- 段落分块：适用于普通文本
- 标题分块：适用于结构化文档
- 语义分块：适用于长文本

## 第三部分：使用示例

```python
from src.document_processors.unified_processor import process

# 处理文本文件
documents = await process(
    "sample.txt",
    chunk=True,
    chunk_strategy='paragraph'
)
```

## 第四部分：总结

文本处理器是一个强大而灵活的工具，能够处理各种文本格式，并提供智能的分块和索引功能。
